# سامانه سامانه پیام (سامانه پیام)

یک پروژه وب ساده و تمیز با Flask که کاربران در آن ثبت‌نام می‌کنند، وارد حساب می‌شوند، پیام دارای موضوع، متن و فایل پیوست می‌فرستند و صندوق دریافتی و ارسالی خود را مشاهده می‌کنند.

## امکانات
- ثبت‌نام و ورود امن با Hash رمز عبور
- ارسال پیام بین کاربران
- صندوق دریافتی، پیام‌های خوانده‌نشده و ارسالی
- فایل پیوست با محدودیت نوع و حجم
- کنترل دسترسی به پیام و فایل
- رابط فارسی، راست‌چین، Responsive و رنگ‌بندی آرام
- دیتابیس SQLite با SQLAlchemy
- Docker و تست اولیه

## اجرای ساده در Windows
1. پوشه پروژه را باز کنید.
2. در نوار آدرس پوشه، `cmd` بنویسید و Enter بزنید.
3. دستورات زیر را اجرا کنید:

```bat
python -m pip install -r requirements.txt
python app.py
```

4. آدرس زیر را در مرورگر باز کنید:

```text
http://127.0.0.1:5000
```

راه ساده‌تر: روی فایل `run.bat` دوبار کلیک کنید.

## روش تست دستی
1. کاربر اول مانند `ali` بسازید.
2. از حساب خارج شوید و کاربر دوم مانند `sara` بسازید.
3. با `ali` وارد شوید و برای `sara` پیام بفرستید.
4. با `sara` وارد شوید و پیام را در Inbox باز کنید.

## اجرای Docker
```bash
docker compose up --build
```
سپس `http://localhost:5000` را باز کنید.

## نکات دفاع
- رمز عبور خام ذخیره نمی‌شود و با Werkzeug هش می‌شود.
- `sender_id` و `receiver_id` کلید خارجی جدول Users هستند.
- نام واقعی فایل از نام ذخیره‌شده جداست تا تداخل نام و حمله Path Traversal کاهش یابد.
- فقط فرستنده و گیرنده اجازه مشاهده پیام و دانلود پیوست را دارند.
- طراحی تا فرم سوم نرمال‌سازی شده و Attachment در جدول جداگانه قرار دارد.
